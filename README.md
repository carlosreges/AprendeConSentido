# AprendeConSentido

Este sitio web está construido como una página estática de presentación para el proyecto educativo **Aprender con Sentido**.

## Descripción

La página fue desarrollada con HTML, CSS y JavaScript puro. Incluye varias secciones para:
- Presentar la propuesta educativa.
- Mostrar servicios de clases particulares, acompañamiento para educadores y transición universitaria.
- Proveer un formulario de contacto funcional en la página principal.
- Contar con un footer consistente y un diseño responsive para dispositivos móviles.

## Estructura del proyecto

- `index.html` — Página principal.
- `clases-particulares.html` — Información sobre clases particulares.
- `contacto.html` — Página de contacto.
- `educadores.html` — Página para educadores.
- `habitar-universidad.html` — Página sobre el acompañamiento universitario.
- `quienes-somos.html` — Página de presentación del equipo y la misión.
- `css/styles.css` — Estilos compartidos y responsive.
- `js/main.js` — Lógica de interacción y comportamiento del sitio.
- `assets/img/` — Imágenes usadas en el sitio.

## Cambios recientes

- Se revisó y limpió el sitio para eliminar estilos inline y mantener la hoja de estilos centralizada.
- Se mejoró la versión móvil con reglas responsive adicionales.
- Se corrigió el footer para que muestre solo el logo sin repetir el nombre del proyecto.
- Se agregó interacción en las tarjetas del bloque "Qué hacemos" para que respondan al hover.

## Cómo usar

1. Abrir cualquiera de los archivos HTML en un navegador.
2. Para editar estilos, modificar `css/styles.css`.
3. Para cambiar el comportamiento interactivo, modificar `js/main.js`.
